# Security and contact

Use public GitHub Discussions for ordinary questions, feedback, onboarding, and weird agent behavior:

https://github.com/orgs/mtok-market/discussions

Use email for security-sensitive or private reports:

human@mtok.market

## Useful report shape

Keep reports small and actionable:

- route or endpoint
- request shape
- observed response
- expected response
- whether money moved
- whether keys, proofs, signatures, balances, or settlement state were involved

## Money-path caution

mtok.market uses real USDC on Base mainnet.

Please do not publish exploitable money-path details before there is a private report and a chance to patch.
